# Mp4Support

Generates MP4 video up to 4K ("capped_4k"), m4a audio only ("audioOnly"), or both for offline viewing.



## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `CAPPED_4K`            | capped_4k              |
| `AUDIO_ONLY`           | audioOnly              |
| `AUDIO_ONLY_CAPPED_4K` | audioOnly,capped_4k    |
